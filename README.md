# 如何在Lightly中构建和运行 React-JavaScript 项目 

## 打开终端

您可以在底部点击【终端】标签打开终端界面

## 构建和运行
输入以下命令来构建和运行项目：

``` bash
# 构建项目
yarn

# 运行项目
yarn dev
```

## 预览项目
项目运行后将会启动服务，系统检测到端口会提示添加，您可以点击左侧文件树 -> “网络”标签 -> 端口预览按钮 打开访问地址。
